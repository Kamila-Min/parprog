# Решение краевой задачи (Простой уровень)

## Компиляция

Linux:

    gcc -fopenmp main.c -o main -lm

## Использование

Расчёт:

    ./main.out {число потоков} {число итераций Метода Ньютона}

Построение графика решения:

    python3 plot.py

Для построения графика необходимы `numpy` и `matplotlib`
